Install Astra on Android

This page exists so installing Astra is less cursed than hunting random assets by hand.

Download latest signed Astra APK Open releases

First install

  1. Download the latest signed app-release.apk.
  2. On your phone, allow installs from your browser or file manager if Android asks.
  3. Open the APK and install Astra.

If you already installed an older build

  • If your current install came from a debug build or a different signing key, Android may refuse to update over it.
  • If that happens, uninstall the old build once, then install the signed Astra APK.
  • After that, future signed builds should update normally as long as the signing key stays the same.

What this download page does

The main download page checks GitHub releases and redirects to the newest release asset named app-release.apk, including prereleases.
