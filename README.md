## Repo layout
- `astra-wakeup-android/` — current Android app project for Astra
- `astra-wakeup-mvp/` — older MVP bits
- `docs/` — GitHub Pages download/install page

## Signed APK behavior
The GitHub Pages site redirects to the newest release asset named `app-release.apk`, including prereleases.
That gives you one stable link even when GitHub’s normal “latest release” behavior is being annoyingly selective.
